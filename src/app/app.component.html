What is TypeScript?

  • A superset of JavaScript
  • TypeScript transpiles to JavaScript
  • Adds optional static typing to JavaScript
  • Adds class-based object-oriented programming to JS
  • Client-side or server-side development
  • Designed for development of large applications
  • Developed and maintained by Microsoft
  • Free and open-sourced
  • Google added as an approved language

    • C/C++, Java, JavaScript*, Python, Go,
    TypeScript & Kotlin

Why use TypeScript?

  • Use future JavaScript features today
  • Easier to organize code base in large/complicated apps
  • Detect errors at dev time
  • Intellisense
  • Enhanced tooling - Language Services
  • Recommended by Angular
  • Majority of Angular documentation uses TypeScript
  • If you already use staticly-typed languages, it is easier to pick up
  • Learning statically-typed skills will be transferable skills
  • Lets you use things that ECMA Script may not add
